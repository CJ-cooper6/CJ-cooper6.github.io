Canvas

Canvas 可以通过HTML、CSS、JS动态设置 三种方法改变画布的大小，但如果使用 CSS 来设置宽高的话，画布就会按照 300 * 150 的比例进行缩放。所以尽量使用 HTML 的width 和 height 属性或者直接使用 JS 动态来设置宽高，不要使用 CSS 设置。
